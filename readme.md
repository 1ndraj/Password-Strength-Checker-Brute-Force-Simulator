# 🔐 Password Security Tool

## 📌 Description
This project demonstrates password strength analysis and brute-force attack simulation.

## 🚀 Features
- Password strength checker
- Brute-force attack simulator
- Demonstrates weak password risks

## 💣 Demo
Weak passwords like "admin123" can be cracked instantly using a wordlist.

## 🛡️ Learning Outcomes
- Importance of strong passwords
- How brute-force attacks work
- Basic defensive security concepts

## 🛠️ Tech Used
- Python
